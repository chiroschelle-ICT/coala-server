# Instructions

## Info
- uses nodejs server
- Poort 3000
- Zet xampp Sql en apache aan voor lokale server
- Need body-parser to give data from service to api

## Installs:
### Npm modules
``` npm install ```

### body parser
``` npm i body-parser ```

### Cors
``` npm i cors ```

### Express
``` npm i express ```

### Mysql
``` npm i mysql ```

## Start server:
``` node server.js ```
